# 給油配送 分析ダッシュボード

灯油配送業務の効率分析ツールです。給油情報と顧客情報をアップロードすると、グループ・エリア別の配送効率ダッシュボードを自動生成します。

## 使い方

1. `index.html` をブラウザで開く（または Webサーバーで配信）
2. `@zero-spec.com` のGoogleアカウントでログイン
3. 2つのファイルをアップロードして「分析実行」

> **デモデータで動作確認したい場合** は「デモデータで実行」ボタンを押す

---

## 必要なファイル

### FILE 01 — 給油情報（xlsx / csv）

| 列 | 内容 |
|---|---|
| A列 | 顧客番号 |
| B列 | 配送日 |
| C列 | 配送量（L） |

### FILE 02 — 顧客情報（xlsx）

以下の2シートを含むExcelファイル：

- **顧客登録** シート：顧客番号・グループ・エリア・タンク種別・しきい値など
- **タンクタイプリスト** シート：タンク種別と容量の対応表

```
    ├── tables.js       # テーブル描画

```

---

## 設定変更

`js/config.js` を編集するだけで認証設定を変更できます：

```js
const CONFIG = {
  clientId: 'YOUR_GOOGLE_CLIENT_ID',
  domain:   'your-company.com',  // ← ログイン許可ドメイン
};
```

Excelのシート名を変更した場合は `SHEET_NAMES` も合わせて変更してください。

## 主要指標の定義

| 指標 | 定義 |
|---|---|
| しきい値配送達成率 | 給油量 ÷ (タンク補正容量 − しきい値)。100%超 = 高効率配送 |
| 給油時在庫率 | 給油直前の残量 ÷ タンク補正容量 |
| グラフ傾き | 1日あたりの消費率（達成率中央値 × しきい値率 ÷ 平均配送間隔） |
